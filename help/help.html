Instructions

Menu

The menu lets the user start a new game, restart a current game from the beginning or start a specific game with preset tile placement The user can also undo their recent moves back to the beginning of the game. The user can also see a list of the cleared tiles from their current game. The user may also toggle the sound on and off.

Game board

The Mah Jong gameboard consist of tiles arranged according to the rules of Maj Jong. Tiles are played by clicking on them.


Objective

The goal is to select matching tiles until the board has been completely cleared. The game is won by clearing the board of all tiles.
